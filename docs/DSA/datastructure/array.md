# 数组

数组是可以存储多个相同元素的结构，在内存中分配的区域是连续的，大多数数据结构使用数组来实现它们的算法。

> 元素：数组中存储的每个项目都称为元素  
> 索引：数组中元素的每个位置都有一个数字索引，每个元素通过索引进行访问，索引从 0 开始。

## 优点：
  
> 1、按照索引查询元素速度快  
> 2、按照索引遍历数组方便

## 缺点：
  
> 1、数组的大小固定后就无法扩容了  
> 2、数组只能存储一种类型的数据  
> 3、添加，删除的操作慢，因为要移动其他的元素

## 适用场景：
  
> 频繁查询，对存储空间要求不大，很少增加和删除的情况
